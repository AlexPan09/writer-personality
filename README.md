# XZTI 写作者量表测试

一款专为写作者设计的性格测试网页应用。

## 本地运行

```bash
cd writer-personality
python -m http.server 8080
```

然后在浏览器中访问 http://localhost:8080/

## 项目结构

```
writer-personality/
├── index.html          # 主页面
├── css/style.css       # 样式文件
├── js/app.js           # 业务逻辑
├── js/vendor/          # 第三方库
│   └── qrcode-generator.js
├── data/questions.json # 题目数据
└── asset/              # 人格形象图片
```
